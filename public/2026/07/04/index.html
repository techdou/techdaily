Mistral AI 发布开源形式化验证模型 Leanstral 1.5

Mistral AI发布开源代码Agent模型Leanstral 1.5，面向Lean 4形式化验证，采用Apache 2.0许可，已在HuggingFace开放并提供免费API端点。

Mistral AI发布Leanstral 1.5，一款面向Lean 4的开源代码Agent模型，采用MoE架构，总参数量119B、激活参数约6B至6.5B，支持256k上下文长度与多模态输入，隶属Mistral Small 4系列。官方数据显示该模型在miniF2F上达到100%、PutnamBench上解决587/672题、FATE-H上达87%、FATE-X上达34%。模型权重以Apache 2.0许可发布于HuggingFace，同时提供免费API端点leanstral-1-5，官方建议通过Mistral Vibe使用，也支持通过vLLM本地部署。

模型发布

生数科技发布 Vidu S1 实时交互模型，已开启内测

生数科技正式发布 Vidu S1 实时交互模型，支持语音控制角色行为与无限时长连续互动，提供 540P 实时视频生成，目前已开启内测。

生数科技在 2026 全球数字经济大会上发布 Vidu S1 实时交互模型。该模型支持实时视频通话和语音控制角色行为，采用自回归扩散模型实现无限时长的连续互动，用户仅需上传一张图片即可创建支持自定义音色的交互角色。Vidu S1 可提供 540P 分辨率及 25FPS（最高 42FPS）的实时生成能力，目前已开启网页端、API 和 APP 端内测。

TRAE 推出限时福利 速通付费用户可获专属速通权益

TRAE 宣布即日起向速通付费用户自动发放 50 次 TRAE Work 专属速通权益，此限时活动结束时间待定。

TRAE 宣布即日起为其速通付费用户提供限时福利。符合条件的当前生效及新开通用户将由系统自动发放 50 次 TRAE Work 专属速通权益。该权益自发放起 31 天内有效，不可转赠或退款。通过 AI 创造力大赛获赠速通 Pro 的用户不享受此福利，且活动随时可能结束。

元宝搜索升级 回答现可自动配图讲解

腾讯元宝搜索功能完成升级，现可在回答中自动生成并搭配图解，以图文结合方式呈现信息，使复杂问题的回答更直观易懂。

腾讯元宝对其搜索功能进行了升级，核心变化在于回答结果中新增了自动图解能力。升级后，当用户提出问题时，元宝在文字回答之外还会自动搭配图解，以图文结合的方式展示信息。该功能旨在使复杂问题的呈现更加清晰直观。目前该升级已在元宝搜索中上线可用。

豆包宣布智能体功能将于2026年7月中旬下线

豆包通知因产品调整，其智能体功能将于2026年7月15日下线，建议提前备份数据，相关数据将于10月中旬按隐私政策处理。

豆包发布通知宣布，因产品功能调整，其智能体功能将于2026年7月中旬下线。功能下线后，用户仍可在一段时间内查看并保存智能体信息及历史对话数据。官方建议用户通过截图或分享导出文本等方式提前完成数据备份。至2026年10月中旬，豆包将根据《隐私政策》对相关数据进行处理，届时数据将无法查看或恢复。

技术与洞察

Anthropic 工程师分享 Claude Fable 5 编码指南：聚焦消除“未知项”

Anthropic工程师近日发文分享了Claude Fable使用指南，提出开发者应通过理清已知与未知来提升AI编程效果，并介绍了盲点扫描、头脑风暴等多种实用技巧。

近日，Anthropic工程师@trq212在X平台发布长文，分享了他使用Claude Fable 5进行智能编程的实践经验。文章核心观点认为，用户提供的提示和上下文如同“地图”，而实际代码库与现实约束则是“领土”，二者间的差异即“未知”；Fable是首个让他感到工作质量取决于澄清未知能力的模型。为此，他总结了在实现前、中、后多个阶段发现未知的技巧，包括盲点扫描、头脑风暴、访谈、源码参考、实现计划与笔记等，并以Fable发布视频的全自动剪辑为例展示了实际应用。

快手可灵AI确定获190亿元融资 阿里腾讯百度参投

快手在港交所发布公告，旗下可灵AI已确定获得190.48亿元融资，总增资款项上限为204.47亿元，阿里、腾讯及百度等均有参投。

快手在港交所发布公告，宣布旗下视频生成业务可灵AI将完成一笔上限为204.47亿元的融资，目前已确定金额为190.48亿元。增资完成后快手持股比例降至约68.33%，阿里、腾讯、百度及多家国资参投；北京可灵将作为主体持有相关业务，快手管理层给出的交易前估值约为150亿美元。同时，北京可灵向关键人员授出股权奖励及期权，另据《南华早报》报道称，快手预计将在未来12个月内启动可灵AI赴港上市流程。

Cherry Studio 开启 V2 版本内测

Cherry Studio 宣布正式开启 V2 版本内测。该版本将 Agent 变为核心并重写了底层数据系统，带来性能提升、多窗口支持及 Agent 模型端点限制解除等更新。

Cherry Studio 官方宣布正式开启 V2 版本内测，对底层数据和任务系统进行全面重写。新版本将 Agent 变为核心，支持任意 OpenAI 兼容端点接入，并新增 HTML 实时预览卡片、多窗口及独立窗口等功能。目前该版本处于持续变化状态，部分页面与 V1 功能仍在迁移并可能存在 BUG。用户现可通过官方内测群或开源仓库参与测试与反馈。
