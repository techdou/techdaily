Anthropic 暂停 claude -p 和 Claude Agent SDK 等用途进行独立计费的变更

Anthropic 官方宣布暂停原定近日生效的 Claude Agent SDK 独立计费变更。目前相关工具仍按原订阅配额计算，官方表示正在更新方案以更好地支持开发者。

根据社区讨论，Anthropic 官方向用户发布邮件，宣布暂停了原定于近日生效的 Claude Agent SDK、claude-p 及第三方应用专属月度额度计费变更。

目前相关工具的用量仍完全按照原订阅配额计算，订阅限额不变，用户无需领取新额度。

官方表示正在努力更新方案以更好地支持开发者，新的变更生效前会提前通知。

此外，有非官方观点猜测此次暂停或与当前的舆论压力有关。

字节跳动发布 Seedance-2.0-mini 视频生成模型

字节跳动正式上线 Seedance-2.0-mini 高性价比视频生成模型。生成成本较 Seedance 2.0 降低约50%，最高支持720P分辨率，已在即梦、剪映及 BytePlus 等平台开放体验。

字节跳动正式上线 Seedance-2.0-mini 高性价比视频生成模型。

该模型支持参考图片、视频、音频生成视频、视频编辑、视频延长及首尾帧生视频等功能。输出分辨率为 480P 和 720P，时长为 4 至 15 秒，帧率为 24fps。

火山方舟官方称该模型生成成本较 Seedance 2.0 降低约 50%。定价为包含视频输入 14 元/百万tokens、不含视频输入 23 元/百万tokens。

官方称其较 Seedance 2.0 成本低约 50%，速度为 Seedance 2.0 Fast 的两倍，输出效果与 Fast 相当。

该模型已在即梦、剪映及 BytePlus 等平台开放体验。火山方舟 API 预计6月22日开放调用。

Kimi K2.7 Code 高速版发布 输出速度提升至6倍

Kimi 上线 K2.7 Code 高速版模型，常规编程场景下输出速度约为普通版的5-6倍。目前已向 API 开发者及部分会员开放，预计7月起陆续向 Allegretto 及以上会员开放。

Kimi 官方宣布 K2.7 Code 高速版模型已上线，目前已向 Kimi API 开发者、Kimi Business 用户及加入抢先体验计划的用户开放。

该高速版与普通版基于同一模型，常规编程场景下输出速度约为普通版的 5-6 倍。

模型发布

微软开源代码探索模型 FastContext-1.0

微软开源 FastContext-1.0 模型家族，专为 LLM coding agent 打造。作为 subagent 通过只读工具返回精准上下文，最高可降低主 Agent 60% 的 Token 消耗。

微软于近日开源了 FastContext-1.0 模型家族，这是一款专为 LLM coding agent 打造的轻量级代码库探索模型。

该模型的核心用途是作为主 Agent 的 subagent 被按需调用。

它通过并行发出 READ、GLOB 和 GREP 三种只读工具调用来定位代码，最终返回紧凑的文件路径和行范围作为聚焦上下文。

这一设计将代码探索与问题解决这两个任务进行了有效分离。

官方提供的数据显示，将其集成到 Mini-SWE-Agent 中，不仅能将端到端解决率最高提升 5.5%，还能大幅减少主 Agent 的 Token 消耗，降幅最高可达 60%。

该模型在 MIT 许可证下开源。

DevEco Code 和 DevEco CLI 开源，提供鸿蒙原生 AI 编程能力

华为发布 DevEco Code 与 DevEco CLI 开发工具。DevEco Code 提供开箱即用的 Agent 能力，内置 GLM-5.1 免费模型通道；DevEco CLI 将鸿蒙开发工具链封装为统一命令行入口。

华为发布开发工具 DevEco Code 与DevEco CLI。

DevEco Code 是一款基于 OpenCode 扩展的 AI Agent 工具，支持自主完成代码编写、编译构建与真机调试等全流程，并内置 GLM-5.1 免费模型通道与 Build、Plan、Goal 三种工作模式。

DevEco CLI 将 HarmonyOS 开发工具链封装为统一命令行入口，用户只需一行 init 命令即可将其 Skill 或 MCP 服务集成到其他外部 Agent 中。

两款工具目前均已在 GitCode 上开源，支持 Windows 和 macOS 平台。
