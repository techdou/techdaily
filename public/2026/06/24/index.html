OpenAI 联合 Broadcom 推出自研 AI 芯片 Jalapeño

OpenAI 联合博通推出首款自研AI芯片Jalapeño，专为大型语言模型推理设计，历时九个月完成开发。目前芯片仍在测试，官方数据显示其性能功耗比优于现有方案。该芯片计划于二零二六年底部署，将支持ChatGPT和Codex等产品，以降低运营成本。

OpenAI 联合 Broadcom 推出其首款自研 AI 芯片 Jalapeño，这是一款专门针对大型语言模型推理工作负载定制的加速器。该芯片从零开始设计，历时九个月完成流片，期间 OpenAI 的自有模型被用于加速设计与优化流程。虽然芯片仍在测试阶段，但官方称初期数据显示其性能功耗比显著优于当前最先进的替代方案。Jalapeño 计划于 2026 年底进行初步部署，旨在支持 ChatGPT、Codex 及 API 等产品，并扩展 OpenAI 的全栈基础设施。

豆包专业版发布 推出 68 元起阶梯订阅及大学生特惠

豆包正式上线专业版服务，推出基于豆包 2.1 系列大模型的全新办公任务模式，支持操作本地电脑、浏览器与调用 Skills 技能。免费用户现已可体验接入 2.1 Turbo 模型的该模式，专业版用户则可使用 2.1 Pro 模型并享有更高额度，目前提供连续包月 68 元至 500 元的三档订阅套餐，即将上线针对大学生的教育优惠方案。

豆包正式推出专业版服务，基于最新的豆包 2.1 系列大模型，为专业用户处理复杂生产力场景提供更高使用额度及全新方案。此次上线的办公任务模式支持在用户授权下操作本地电脑与浏览器、执行定时任务、调用首批 Skills 技能包以及部署网站应用。免费用户可体验接入 2.1 Turbo 模型的办公任务模式，专业版用户则接入 2.1 Pro 模型，官方提供 68 元、200 元与 500 元的三档连续包月套餐。此外，官方即将推出针对大学生的 6 个月特惠活动，并为视障人群提供专属优惠方案。

阿里发布开源首个语言世界模型Qwen-AgentWorld

阿里千问团队发布原生语言世界模型 Qwen-AgentWorld，该模型专为 AI Agent 研发，单一模型同时覆盖文本类与 GUI 类共七大智能体交互环境。该模型含{2|两}个版本，目前已开放配套评测基准 AgentWorldBench 以及 35B-A3B 版本模型权重。

阿里千问团队发布了自称首个原生语言世界模型 Qwen-AgentWorld，并提供 35B-A3B 与 397B-A17B 两种规模，目前官方已开源 35B-A3B 版本模型权重及配套评测基准 AgentWorldBench。该模型专为 AI Agent 研发，将环境建模贯穿继续预训练、监督微调到强化学习的全流程，单一模型即可同时覆盖文本类与GUI类共七大智能体交互环境。官方称，Qwen-AgentWorld 并非为了替代真实环境，而是作为互补路径增强通用智能体的决策能力。

Google 推出 Gemini 3.5 Flash 内置 Computer Use 功能

Google 官方宣布，Gemini 3.5 Flash 已支持 Computer Use 功能，现已通过 Gemini API 等渠道开放。开发者可借此构建跨浏览器、移动端和桌面环境操作的自定义 Agent

Google DeepMind 宣布将 Computer Use 作为 Gemini 3.5 Flash 的内置工具正式推出，该功能此前仅作为独立的 Gemini 2.5 计算机使用模型提供。开发者现可使用 3.5 Flash 构建能在浏览器、移动端和桌面环境中查看、推理和执行操作的自定义 Agent，用于持续软件测试等长程和企业自动化任务。该功能已通过 Gemini API 和 Gemini Enterprise Agent Platform 向开发者和企业开放。

Notion 推出 External Agents 功能，接入 Claude 与 Cursor

Notion 官方宣布推出 External Agents 功能，用户可在 Notion 内直接向智能体分配任务并协同工作，首批支持 Claude 和 Cursor。

Notion 官方宣布推出 External Agents（外部智能体）功能，首批支持 Claude 和 Cursor。用户可在 Notion 内直接向智能体分配任务并协同工作。根据官方帮助文档，Claude 智能体目前处于 Beta 阶段，面向 Business 和 Enterprise plans，通过 Notion credits 按运行次数计费，不支持个人 Anthropic 账户。而 Cursor 官方博客指出，Notion 借助 Cursor SDK 完成集成，用户可通过配置 MCP 服务器等方式定制 Cursor，让其端到端完成代码规划、测试并发起 PR。Notion 还表示未来将引入更多智能体，并支持企业连接自建的智能体。

Figma 宣布在画布中原生推出多项新工具。其中 Figma Motion 能将时间轴与关键帧带入设计文件，并支持通过 Figma Agent 生成动画及在 Dev Mode 中导出代码。此外，官方还预告了代码层、3D 变换等即将推出的新功能。

Figma 官方宣布推出 Figma Motion，将动画时间轴、关键帧和预设直接整合进设计画布，目前该功能已进入公开测试阶段。用户可利用 Figma Agent 根据提示生成关键帧，且动画支持导出为 MP4 等格式或在 Dev Mode 中复制 CSS、JSON 及 React 代码，并兼容 MCP。同时，官方还推出了由 Agent 生成的 Shader 效果、生成式插件和 Weave 工具，并预告了即将于 7 月推送的代码层、3D 变换等新功能。Motion 功能的可用性因账户类型而异，其中 Starter 用户支持受限导出，付费计划用户则可使用完整系统集成及 Figma Agent 功能。

英国政府资助牛津大学和 UCL 建立下一代 AI 研究实验室

近日，英国政府宣布提供最高 6000 万英镑资金，在牛津大学和伦敦大学学院设立两个新的 AI 研究实验室。官方称，此举旨在开发下一代成本更低、更可靠且易于企业和公共服务采用的开源 AI 技术。

英国政府宣布将在未来 6 年内提供最高 6000 万英镑，在牛津大学和伦敦大学学院设立两个新的 AI 研究实验室。这两个实验室将分别专注于开发可在普通硬件上运行的新型开源 AI 技术以及重新思考 AI 的高效学习与适应方式。官方表示，此举旨在降低先进 AI 工具的运行成本与使用门槛，使更多英国企业和公共服务能够采用这些技术。此外，政府还将为实验室提供大规模计算资源支持。
